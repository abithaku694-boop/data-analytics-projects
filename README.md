Restaurant Data Analysis Dashboard (Power BI)
Developed an interactive Power BI dashboard to analyze restaurant data and uncover insights on customer behavior, cuisine trends, and city-wise performance.
Analyzed 9,500+ restaurants and 1M+ votes
Identified top cities like New Delhi, Gurgaon, and Noida
Highlighted popular cuisines (North Indian, Chinese)
Evaluated average rating and cost trends
Created dynamic visuals and KPI metrics for decision-making
